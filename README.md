# BrowserCraft Multiplayer (P2P)

Браузерная Minecraft-like игра с 3D-камерой, текстурами, чанковой генерацией и P2P-мультиплеером через WebRTC (без центрального сервера для игрового трафика).

## Запуск

```bash
npm install
npm start
```

Открой в браузере: `http://localhost:3000`

## Меню Host / Connect (без npm у игроков)

- **Host**: создаёт код хоста.
- **Connect**: принимает код хоста, генерирует код игрока.
- Хост вставляет код игрока и подтверждает подключение.

Поток подключения:

1. Игрок A: **Host** → копирует код хоста → отправляет игроку B.
2. Игрок B: **Connect** → вставляет код хоста → получает код игрока.
3. Игрок B отправляет код игрока игроку A.
4. Игрок A вставляет код игрока → **Подключить игрока**.
5. Оба нажимают **Закрыть и играть**.

## Управление

- `W / A / S / D` — движение
- `Space` — прыжок
- `Shift` — спринт
- Мышь — камера
- `ЛКМ` — ломать блок
- `ПКМ` — ставить блок
- `1..5` — выбор блока
